Chương 3: Làm việc với CSS

I. Khái niệm Cơ bản

CSS (Cascading Style Sheets) là ngôn ngữ quy định phong cách trình bày cho các tài liệu HTML. CSS giúp tách biệt nội dung và hình thức, tăng khả năng tái sử dụng và quản lý code hiệu quả.

  • Mục đích: Quản lý, thay đổi giao diện website nhanh chóng, đồng nhất.
  • Nguyên tắc Cascading: Khi có nhiều quy tắc ảnh hưởng cùng một phần tử, CSS sẽ áp dụng theo thứ tự ưu tiên từ cao đến thấp: Inline > Internal > External > Mặc định trình duyệt.

Cú pháp CSS cơ bản

selector {
    property: value;
    property2: value2;
}

Ví dụ:

h1 {
    color: blue;
    font-size: 32px;
}

II. Các Phương thức Nhúng CSS

  • 1. Inline Style (CSS nội tuyến): Ghi trực tiếp trong thuộc tính style của thẻ HTML. Ưu tiên cao nhất nhưng khó quản lý.
  • 2. Internal/Embedded Style (CSS nội bộ): Đặt trong thẻ <style> ở phần <head> của HTML.
  • 3. External Style Sheet (CSS ngoài): Liên kết đến file .css bằng thẻ <link>. Phương pháp tốt nhất cho dự án lớn.
Ví dụ Inline:
<p style="color: red; font-weight: bold;">Đoạn văn này có màu đỏ.</p>
Ví dụ Internal:
<head>
    <style>
        body { background-color: lightgray; }
        .box { border: 1px solid black; }
    </style>
</head>
Ví dụ External:
<head>
    <link rel="stylesheet" href="style.css">
</head>

2. Bộ chọn kết hợp

  • Con cháu: A B – Chọn mọi B nằm trong A.
  • Con trực tiếp: A > B – Chọn B là con trực tiếp của A.
  • Kề cận: A + B – Chọn B ngay sau A.
  • Anh em chung cấp: A ~ B – Chọn tất cả B sau A cùng cấp cha.

3. Bộ chọn giả lớp (Pseudo-class) và giả phần tử (Pseudo-element)

  • Giả lớp: :hover, :focus, :nth-child(n)
  • Giả phần tử: ::before, ::first-line
Ví dụ tổng hợp:
<div id="container">
    <h1>Tiêu đề chính</h1>
    <p class="intro">Đoạn giới thiệu 1.</p>
    <ul><li>Mục 1</li><li>Mục 2</li></ul>
    <p>Đoạn giới thiệu 2.</p>
</div>
<a href="#" class="button">Nhấn vào đây</a>
style.css:
#container { border: 2px solid gray; }
.intro { color: purple; }
#container p { font-family: Arial, sans-serif; }
ul > li { list-style-type: square; }
.button:hover { background-color: lightgreen; cursor: pointer; }

IV. Các Thuộc tính CSS

1. Màu sắc & Hình nền

  • color: Màu chữ (vd: red, #FF0000, rgb(255,0,0))
  • background-color: Màu nền
  • background-image: Ảnh nền
  • background-position, background-repeat

2. Font & Văn bản

  • font-family: Kiểu chữ
  • font-size: Cỡ chữ
  • font-weight: Độ đậm
  • text-align: Căn lề văn bản
  • text-decoration: Trang trí chữ (gạch chân, gạch ngang...)
  • line-height: Khoảng cách dòng

3. Hộp và bố cục (Box Model)

  • width, height: Kích thước hộp nội dung
  • padding: Đệm bên trong
  • border: Viền hộp
  • margin: Lề ngoài
  • box-sizing: Quy định cách tính kích thước hộp (content-box, border-box)
Ví dụ Box Model:
.box {
    width: 300px;
    height: 100px;
    border: 2px solid #333;
    padding: 20px;
    margin: 20px auto;
    background-color: #ffffe0;
    box-sizing: border-box;
}

4. Hiển thị (display) & Vị trí (position)

  • display: block | inline | inline-block | none
  • position: static | relative | absolute | fixed
Ví dụ thuộc tính phổ biến:
p {
    color: green;
    background-color: #f5f5f5;
    font-size: 18px;
    border-bottom: 1px dotted #ccc;
    text-align: justify;
}

V. Layout cơ bản và Nâng cao

1. Thuộc tính display

  • block: Chiếm toàn bộ dòng (vd: div, p, h1...)
  • inline: Không xuống dòng mới (vd: span, a...)
  • inline-block: Kết hợp ưu điểm block và inline
  • none: Ẩn phần tử khỏi trang

2. Flexbox - Bố cục linh hoạt hiện đại

  • display: flex; trên phần tử cha
  • flex-direction, justify-content, align-items điều khiển hướng sắp xếp, căn chỉnh

VI. Bài tập Thực hành

  1. Thanh điều hướng ngang (Navigation Bar):
    • Tạo <ul><li> làm menu. Sử dụng Flexbox hoặc display: inline-block để nằm ngang.
    • Loại bỏ dấu chấm đầu dòng (list-style-type: none;).
    • Căn đều các mục menu (justify-content: space-around;).
    • Hiệu ứng :hover cho liên kết.
    <ul class="navbar">
    <li><a href="#">Trang chủ</a></li>
    <li><a href="#">Giới thiệu</a></li>
    <li><a href="#">Liên hệ</a></li>
</ul>

    .navbar {
    display: flex;
    list-style-type: none;
    justify-content: space-around;
    background: #3498db;
    padding: 0;
}
.navbar li a {
    color: white;
    text-decoration: none;
    padding: 10px 20px;
    display: block;
    transition: background 0.2s;
}
.navbar li a:hover {
    background: #217dbb;
}
  2. Thử thách Box Model:
    • Tạo hộp với width: 400px, border: 5px solid #333, padding: 20px, margin: 20px auto, background-color: lightyellow, box-sizing: border-box.
    <div class="mybox">Đây là hộp sử dụng Box Model</div>

    .mybox {
    width: 400px;
    border: 5px solid #333;
    padding: 20px;
    margin: 20px auto;
    background-color: lightyellow;
    box-sizing: border-box;
}
  3. Bộ chọn và Ưu tiên:
    • Tất cả <p> màu xanh lá (green)
    • <p class="important"> màu xanh dương (blue)
    • <p id="urgent"> màu đỏ (red) – ID có ưu tiên cao nhất
    <p>Đoạn thường</p>
<p class="important">Đoạn quan trọng</p>
<p id="urgent" class="important">Đoạn khẩn cấp</p>

    p { color: green; }
p.important { color: blue; }
#urgent { color: red; }

    Kết quả: Đoạn #urgent luôn màu đỏ do ID ưu tiên cao nhất.
