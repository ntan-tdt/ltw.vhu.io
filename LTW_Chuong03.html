Chương 3: Làm việc với CSS

I. Lý thuyết

CSS (Cascading Style Sheets) là ngôn ngữ dùng để định dạng và trình bày nội dung HTML. CSS giúp kiểm soát màu sắc, bố cục, font chữ, khoảng cách và nhiều yếu tố giao diện khác.

1. Cú pháp CSS

selector {
    property: value;
}

Ví dụ:

p {
    color: blue;
    font-size: 16px;
}

2. Cách áp dụng CSS vào HTML

  • Inline: Dùng thuộc tính style trong thẻ HTML
  • Internal: Viết trong thẻ <style> trong phần <head>
  • External: Liên kết đến file .css bên ngoài bằng <link>

3. Một số thuộc tính CSS phổ biến

  • Màu sắc: color, background-color
  • Font chữ: font-family, font-size, font-weight
  • Bố cục: margin, padding, width, height
  • Viền: border
  • Canh lề: text-align, vertical-align

II. Thực hành

Bài 1: Đổi màu chữ và nền

<p style="color: white; background-color: green;">
    Xin chào CSS!
</p>

Bài 2: Tạo khung viền và căn giữa

<div style="border: 2px solid blue; text-align: center;">
    Nội dung căn giữa và có viền
</div>

Bài 3: Sử dụng file CSS bên ngoài

<head>
    <link rel="stylesheet" href="style.css">
</head>

<p class="highlight">Đoạn văn nổi bật</p>

style.css:

.highlight {
    color: red;
    font-weight: bold;
}

Bài 4: Tạo bố cục 2 cột bằng float

<div style="width: 50%; float: left;">Cột trái</div>
<div style="width: 50%; float: right;">Cột phải</div>

Bài 5: Hiệu ứng hover

<style>
button:hover {
    background-color: orange;
    color: white;
}
</style>

<button>Di chuột vào tôi</button>
