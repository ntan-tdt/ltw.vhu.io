Chương 4: Lập trình với Javascript

1. Giới thiệu

Javascript là ngôn ngữ lập trình phía trình duyệt, giúp tạo ra các tương tác động trên trang web như xử lý sự kiện, thay đổi nội dung, kiểm tra dữ liệu đầu vào...

2. Cú pháp cơ bản

Javascript sử dụng cú pháp tương tự như C/C++ và Java. Một số cú pháp cơ bản:

// In ra màn hình
console.log("Xin chào Javascript");

// Khai báo biến
let x = 10;
const PI = 3.14;

// Câu điều kiện
if (x > 5) {
    console.log("x lớn hơn 5");
}

3. Biến và kiểu dữ liệu

Các kiểu dữ liệu phổ biến: Number, String, Boolean, Array, Object.

let name = "Văn Hiến";
let age = 20;
let isStudent = true;
let scores = [8, 9, 10];
let student = { name: "An", age: 21 };

4. Hàm

Hàm là khối mã thực hiện một nhiệm vụ cụ thể.

function greet(name) {
    return "Xin chào " + name;
}
console.log(greet("Lan"));

5. Xử lý sự kiện

Javascript có thể xử lý các sự kiện như click, nhập liệu, di chuột...

DOM cho phép Javascript truy cập và thay đổi nội dung HTML.
