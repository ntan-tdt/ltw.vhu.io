Chương 4: Lập trình với Javascript

1. Các khái niệm cơ bản

JavaScript (JS) là ngôn ngữ lập trình kịch bản chủ yếu chạy phía trình duyệt, giúp làm cho trang web động, tương tác và hấp dẫn hơn. Đây là lớp thứ ba (hành vi - behavior) trong bộ ba HTML (nội dung) - CSS (trình bày) - JS (hành vi).

  • Chạy phía client: JS chủ yếu chạy trên trình duyệt người dùng.
  • Thông dịch: Không cần biên dịch, code thực thi trực tiếp.
  • Kiểu động: Không cần khai báo kiểu dữ liệu khi khai báo biến.

Cách nhúng JavaScript vào HTML

  • Nội tuyến (Inline): Thuộc tính sự kiện trong thẻ HTML.
    Ví dụ: <button onclick="alert('Hello!')">Click</button>
  • Nội bộ (Internal): Đặt mã JS trong thẻ <script> (thường ở cuối <body>).
  • Bên ngoài (External): Liên kết file .js bên ngoài bằng <script src="app.js"></script> (nên dùng cho dự án thực tế).
Ví dụ nhúng JS nội bộ:
<script>
    alert('Chào mừng đến với Javascript!');
</script>

2. Các Khái niệm Lập trình Cốt lõi

Biến, Hằng số và Kiểu dữ liệu

  • let: Khai báo biến (có thể gán lại giá trị, phạm vi khối).
  • const: Khai báo hằng số (không gán lại giá trị).
  • var: Khai báo biến kiểu cũ (ít dùng).
const PI = 3.14;
let ten = "Lan";
let tuoi = 20;
let isStudent = true;
let arr = [1, 2, 3];
let obj = { name: "An", age: 21 };

Toán tử và Biểu thức

  • Số học: +, -, *, /, %, ** (lũy thừa)
  • So sánh: ==, ===, !=, >, <, >=, <=
  • Logic: && (và), || (hoặc), ! (phủ định)
let x = 10;
if (x > 5) {
    console.log("x lớn hơn 5");
}

Cấu trúc điều khiển

  • Điều kiện: if, else if, else, switch-case
  • Vòng lặp: for, while, do...while
for (let i = 0; i < 3; i++) {
    console.log("Lặp thứ: " + (i+1));
}

3. Hàm (Functions)

Hàm là khối mã thực hiện một nhiệm vụ cụ thể, giúp tổ chức, tái sử dụng code.

  • Function Declaration: function tenHam(thamSo) { ... }
  • Function Expression: const tenHam = function(thamSo) { ... };
  • Arrow Function: const tenHam = (thamSo) => { ... }; (cú pháp hiện đại)
// Hàm khai báo
function greet(name) {
    return "Xin chào " + name;
}
console.log(greet("Lan"));

// Arrow function
const tong = (a, b) => a + b;
console.log(tong(5, 7));

Xử lý sự kiện (Event Handling)

  • Gắn trực tiếp trên HTML (không khuyến khích): <button onclick="myFunc()">
  • Gắn qua thuộc tính JS: element.onclick = myFunc;
  • Nên dùng: element.addEventListener('click', myFunc);
<button id="btn">Bấm vào đây</button>
<p id="msg"></p>
<script>
document.getElementById('btn').addEventListener('click', function() {
    document.getElementById('msg').innerText = "Bạn vừa bấm nút!";
});
</script>

4. Làm việc với DOM

DOM (Document Object Model) là mô hình dữ liệu dạng cây của tài liệu HTML, cho phép Javascript truy cập, thay đổi và thao tác nội dung, thuộc tính, kiểu dáng của trang web.

  • Lấy phần tử: document.getElementById('id'), document.querySelector('.class')
  • Thay đổi nội dung: element.innerHTML / textContent
  • Thay đổi thuộc tính: element.setAttribute('attr', 'value')
  • Thay đổi CSS: element.style.property
  • Thêm/Xóa phần tử: parent.appendChild(child), parent.removeChild(child)
Ví dụ: Thay đổi nội dung và thêm phần tử mới
<div id="box">
  <h3>Danh sách</h3>
  <ul id="list"><li>Mục 1</li></ul>
  <button id="add">Thêm mục mới</button>
</div>
<script>
const list = document.getElementById('list');
document.getElementById('add').onclick = function() {
    const li = document.createElement('li');
    li.textContent = "Mục mới " + (list.children.length + 1);
    list.appendChild(li);
}
</script>

5. Làm việc với Forms

Javascript thường dùng để kiểm tra dữ liệu người dùng nhập trên form trước khi gửi lên server.

  • Sự kiện: submit, input, change
  • Ngăn submit reload trang: event.preventDefault()
  • Truy cập giá trị: input.value
<form id="myForm">
    Tên: <input type="text" id="username"><br>
    Mật khẩu: <input type="password" id="password"><br>
    <button type="submit">Đăng nhập</button>
</form>
<p id="formMsg"></p>
<script>
document.getElementById('myForm').addEventListener('submit', function(e) {
    e.preventDefault();
    let user = document.getElementById('username').value;
    let pass = document.getElementById('password').value;
    let msg = "";
    if (user.trim() === "") {
        msg = "Tên người dùng không được rỗng!";
    } else if (pass.length < 6) {
        msg = "Mật khẩu phải ít nhất 6 ký tự!";
    } else {
        msg = "Đăng nhập thành công!";
    }
    document.getElementById('formMsg').innerText = msg;
});
</script>
